#POO -  Gestão de Academia :
 Um aplicação simples em Java para gerenciar informações de uma academia, utilizando SQLite como banco de dados.

**Funcionalidades**
Cadastro de alunos
Cadastro de professores
Registro de treinos
Consultas e edição de registros

**Tecnologias**
Java 11 ou superior
SQLite
